📌 Overview

A full-stack YouTube Clone prototype inspired by YouTube, built using modern web technologies. The project provides a responsive and user-friendly interface for browsing, searching, and watching videos.

✨ Features
🔐 User Authentication (Login & Signup)
🎥 Video Streaming
🔍 Search Videos
📂 Explore Categories (Trending, Music, Gaming, News, Sports, Learning, etc.)
👍 Like & Dislike Videos
💬 Comment System
📺 Channel & Subscription Management
📜 Watch History
⬇️ Download Videos
💎 Premium Features
🌙 Dark & Light Theme
📱 Responsive Design (Desktop, Tablet & Mobile)
🛠️ Technologies Used
React.js
Node.js
Express.js
MongoDB
HTML5
CSS3
JavaScript
Git & GitHub
🚀 Installation
git clone https://github.com/mynenisahithi/youtube-clone.git
cd youtube-clone
npm install
npm start
